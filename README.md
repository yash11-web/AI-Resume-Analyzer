# AI Resume Analyzer

AI Resume Analyzer is a full-stack web application designed to help job seekers optimize their resumes. By analyzing a resume against a job description, it highlights matching and missing keywords, providing actionable insights to improve the chances of shortlisting in Applicant Tracking Systems (ATS).

---

## Features

- **Resume Upload:** Upload resumes in PDF or DOCX format.
- **Job Description Input:** Paste a job description to compare against your resume.
- **Keyword Analysis:** Identify matching and missing keywords to tailor resumes effectively.
- **Real-Time Results:** Interactive display of analysis results for quick insights.
- **User-Friendly UI:** Clean, responsive interface with intuitive navigation.

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js
- **Database:** Supabase
- **Hosting:** Render
- **AI/Analysis:** Google Gemini API and Custom logic for keyword matching and relevance scoring
