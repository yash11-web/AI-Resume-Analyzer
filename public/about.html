AI Resume Analyzer

About This Project

The AI Resume Analyzer is a web application that helps job seekers improve their resumes. Users can upload a resume and optionally add a job description. The app analyzes the resume, highlights missing keywords, scores it against the job description, and provides actionable tips to improve chances of passing Applicant Tracking Systems (ATS).

This project is built using Node.js and Express for the server, Supabase for user management and data storage, and Google Gemini AI for resume analysis and suggestions.

The goal is to make resume optimization simple, fast, and effective for job applications.
